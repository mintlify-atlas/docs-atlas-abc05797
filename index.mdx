# Welcome to ZIO

ZIO is a zero-dependency Scala library for asynchronous and concurrent programming. Powered by highly-scalable, non-blocking fibers that never waste or leak resources, ZIO lets you build scalable, resilient, and reactive applications that meet the needs of your business.

## Key Features

<AccordionGroup>
  <Accordion title="Lightweight Fibers">
    ZIO's fibers are green threads that are incredibly lightweight and efficient. You can spawn millions of fibers without worrying about system resources.
  </Accordion>
  <Accordion title="Composable Error Handling">
    Handle errors with full type safety. ZIO's error channel lets you track what can go wrong and handle failures gracefully.
  </Accordion>
  <Accordion title="Built-in Dependency Injection">
    Use ZLayer to manage dependencies in a type-safe, composable way without external DI frameworks.
  </Accordion>
  <Accordion title="Powerful Streaming">
    Process infinite streams of data efficiently with ZStream, including backpressure and resource safety.
  </Accordion>
  <Accordion title="Software Transactional Memory">
    Write lock-free concurrent programs using STM primitives like TRef, TMap, and TQueue.
  </Accordion>
  <Accordion title="Comprehensive Testing">
    Test your ZIO applications with built-in test framework, property-based testing, and test services.
  </Accordion>
</AccordionGroup>

## Quick Example

```scala example.scala
import zio._

object HelloWorld extends ZIOAppDefault {
  def run = 
    for {
      _ <- Console.printLine("What is your name?")
      name <- Console.readLine
      _ <- Console.printLine(s"Hello, $name!")
    } yield ()
}
```
