## Overview

A `TestAspect` is an aspect that can be weaved into specs to modify test behavior. You can think of an aspect as a polymorphic function capable of transforming one test into another, possibly enlarging the environment or error type.

## Type Definition

```scala
abstract class TestAspect[+LowerR, -UpperR, +LowerE, -UpperE]
```

## Type Parameters

- `LowerR` - The lower bound on the environment type
- `UpperR` - The upper bound on the environment type
- `LowerE` - The lower bound on the error type
- `UpperE` - The upper bound on the error type

## Core Methods

### `apply` / `@@`

Applies the aspect to every test in a spec.

```scala
def apply[R >: LowerR <: UpperR, E >: LowerE <: UpperE](
  spec: Spec[R, E]
)(implicit trace: Trace): Spec[R, E]
```

**Example:**
```scala
test("my test") {
  assertTrue(true)
} @@ TestAspect.timeout(60.seconds)
```

### `>>>`

Composes two aspects sequentially.

```scala
def >>>[LowerR1 >: LowerR, UpperR1 <: UpperR, LowerE1 >: LowerE, UpperE1 <: UpperE](
  that: TestAspect[LowerR1, UpperR1, LowerE1, UpperE1]
): TestAspect[LowerR1, UpperR1, LowerE1, UpperE1]
```

**Example:**
```scala
val aspect = TestAspect.timeout(10.seconds) >>> TestAspect.flaky
```

## Built-in Aspects

### Execution Control

#### `timeout`

Times out tests using the specified duration.

```scala
def timeout(duration: Duration): TestAspectPoly
```

**Example:**
```scala
test("long running test") {
  ZIO.sleep(5.seconds) *> assertTrue(true)
} @@ TestAspect.timeout(10.seconds)
```

#### `diagnose`

Runs each test on a separate fiber and prints a fiber dump if the test fails or hasn't terminated.

```scala
def diagnose(duration: Duration): TestAspectPoly
```

#### `forked`

Runs each test on its own separate fiber.

```scala
val forked: TestAspectPoly
```

**Example:**
```scala
suite("concurrent tests") {
  test("test1") { ??? } +
  test("test2") { ??? }
} @@ TestAspect.forked
```

### Retry Logic

#### `flaky`

Retries a test until success, with a default limit.

```scala
val flaky: TestAspectPoly
def flaky(n: Int): TestAspectPoly
```

**Example:**
```scala
test("flaky external API call") {
  // test that might fail occasionally
  assertTrue(externalAPI.call())
} @@ TestAspect.flaky(5)
```

#### `retry`

Retries failed tests according to a schedule.

```scala
def retry[R0, E0](schedule: Schedule[R0, TestFailure[E0], Any]): TestAspect[Nothing, R0, Nothing, E0]
```

**Example:**
```scala
test("retry with backoff") {
  assertTrue(unstableTest())
} @@ TestAspect.retry(Schedule.exponential(1.second) && Schedule.recurs(3))
```

#### `eventually`

Retries a test until success, without limit.

```scala
val eventually: TestAspectPoly
```

### Stability Testing

#### `nonFlaky`

Repeats the test a default number of times, ensuring stability.

```scala
val nonFlaky: TestAspectPoly
def nonFlaky(n: Int): TestAspectPoly
```

**Example:**
```scala
test("must pass 100 times") {
  assertTrue(stableOperation())
} @@ TestAspect.nonFlaky(100)
```

#### `repeat`

Repeats successful tests according to a schedule.

```scala
def repeat[R0](schedule: Schedule[R0, TestSuccess, Any]): TestAspectAtLeastR[R0]
```

### Execution Strategy

#### `sequential`

Executes suite members sequentially.

```scala
val sequential: TestAspectPoly
```

**Example:**
```scala
suite("database tests") {
  test("test1") { ??? } +
  test("test2") { ??? }
} @@ TestAspect.sequential
```

#### `parallel`

Executes suite members in parallel.

```scala
val parallel: TestAspectPoly
```

#### `parallelN`

Executes suite members in parallel, up to N concurrent fibers.

```scala
def parallelN(n: Int): TestAspectPoly
```

**Example:**
```scala
suite("parallel tests") {
  // Run up to 4 tests concurrently
  test("test1") { ??? } +
  test("test2") { ??? } +
  test("test3") { ??? } +
  test("test4") { ??? }
} @@ TestAspect.parallelN(4)
```

### Lifecycle Hooks

#### `before`

Runs the specified effect before every test.

```scala
def before[R0, E0](effect: ZIO[R0, E0, Any]): TestAspect[Nothing, R0, E0, Any]
```

**Example:**
```scala
suite("database tests") {
  test("query1") { ??? } +
  test("query2") { ??? }
} @@ TestAspect.before(Database.resetTestData)
```

#### `after`

Runs the specified effect after every test.

```scala
def after[R0, E0](effect: ZIO[R0, E0, Any]): TestAspect[Nothing, R0, E0, Any]
```

#### `beforeAll`

Runs the specified effect once before all tests.

```scala
def beforeAll[R0, E0](effect: ZIO[R0, E0, Any]): TestAspect[Nothing, R0, E0, Any]
```

**Example:**
```scala
suite("integration tests") {
  test("test1") { ??? } +
  test("test2") { ??? }
} @@ TestAspect.beforeAll(startServer)
```

#### `afterAll`

Runs the specified effect once after all tests.

```scala
def afterAll[R0](effect: ZIO[R0, Nothing, Any]): TestAspect[Nothing, R0, Nothing, Any]
```

#### `around`

Evaluates every test between two effects.

```scala
def around[R0, E0](
  before: ZIO[R0, E0, Any], 
  after: ZIO[R0, Nothing, Any]
): TestAspect[Nothing, R0, E0, Any]
```

#### `aroundWith`

Evaluates every test between two effects, where `after` can use the result of `before`.

```scala
def aroundWith[R0, E0, A0](
  before: ZIO[R0, E0, A0]
)(after: A0 => ZIO[R0, Nothing, Any]): TestAspect[Nothing, R0, E0, Any]
```

**Example:**
```scala
suite("tests with transaction") {
  test("test1") { ??? }
} @@ TestAspect.aroundWith(beginTransaction) { tx =>
  rollbackTransaction(tx)
}
```

### Conditional Execution

#### `failing`


**Example:**
```scala

} @@ TestAspect.failing
```

#### `ignore`

Marks tests as ignored.

```scala

```

**Example:**
```scala

} @@ TestAspect.ignore
```

#### `shrinks`

#### `size`

**Example:**

### Environment Management

#### `fromLayer`

Provides each test with a layer.

```scala
def fromLayer[R0, E0](layer: ZLayer[R0, E0, Any]): TestAspect[Nothing, R0, E0, Any]
```

**Example:**
```scala
val dbLayer: ZLayer[Any, Throwable, Database] = ???

suite("database tests") {
  test("query") {
    for {
      db <- ZIO.service[Database]
      result <- db.query("SELECT * FROM users")
    } yield assertTrue(result.nonEmpty)
  }
} @@ TestAspect.fromLayer(dbLayer)
```

#### `fromLayerShared`

Provides all tests with a shared layer.

```scala
def fromLayerShared[R0, E0](layer: ZLayer[R0, E0, Any]): TestAspect[Nothing, R0, E0, Any]
```

## Common Patterns

### Combining Multiple Aspects

```scala
test("complex test") {
  assertTrue(true)
} @@ TestAspect.timeout(30.seconds) 
   @@ TestAspect.flaky(3) 
   @@ TestAspect.tag("integration")
```

### Conditional Aspects

```scala
val dbTests = suite("database tests") {
  test("test1") { ??? } +
  test("test2") { ??? }
} @@ TestAspect.ifEnvSet("DB_URL")
```

### Layer Management

```scala
val tests = suite("service tests") {
  test("test1") { ??? } +
  test("test2") { ??? }
} @@ TestAspect.fromLayerShared(serviceLayer)
```
