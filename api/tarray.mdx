# TArray

`TArray[A]` is a transactional array that wraps an array of `TRef[A]` elements and provides convenient methods for atomic operations on indexed collections.

## Overview

TArray provides transactional access to an array of values. All operations are atomic and can be composed with other STM operations. Unlike regular arrays, TArray operations participate in STM transactions and benefit from automatic retry and rollback.

## Creating TArray

### make

```scala
def make[A](data: A*): USTM[TArray[A]]
```

Creates a new `TArray` initialized with the specified values.

```scala
val createArray: USTM[TArray[Int]] = 
  TArray.make(1, 2, 3, 4, 5)
```

### fromIterable

```scala
def fromIterable[A](data: => Iterable[A]): USTM[TArray[A]]
```

Creates a `TArray` from an iterable collection.

```scala
val createArray: USTM[TArray[String]] = 
  TArray.fromIterable(List("a", "b", "c"))
```

### empty

```scala
def empty[A]: USTM[TArray[A]]
```

Creates an empty `TArray`.

```scala
val emptyArray: USTM[TArray[Int]] = TArray.empty
```

## Element Access

### apply (get element)

```scala
def apply(index: Int): USTM[A]
```

Retrieves the value at the specified index.

```scala
val tx: USTM[Int] = for {
  arr <- TArray.make(10, 20, 30)
  v   <- arr(1) // Gets 20
} yield v
```

### update

```scala
def update(index: Int, fn: A => A): USTM[Unit]
```

Updates the element at the specified index using a function.

```scala
val tx: USTM[Unit] = for {
  arr <- TArray.make(1, 2, 3)
  _   <- arr.update(0, _ * 10) // First element becomes 10
} yield ()
```

### updateSTM

```scala
def updateSTM[E](index: Int, fn: A => STM[E, A]): STM[E, Unit]
```

Updates an element using a transactional function.

```scala
val tx: STM[String, Unit] = for {
  arr <- TArray.make(1, 2, 3)
  _   <- arr.updateSTM(0, n => 
           if (n > 0) STM.succeed(n * 2)
           else STM.fail("negative value")
         )
} yield ()
```


Finds the first or last element matching a predicate.

```scala
val tx: USTM[Option[Int]] = for {
  arr    <- TArray.make(1, 5, 10, 15, 20)
  result <- arr.find(_ > 10) // Some(15)
} yield result
```

### findSTM / findLastSTM

```scala
def findSTM[E](p: A => STM[E, Boolean]): STM[E, Option[A]]
def findLastSTM[E](p: A => STM[E, Boolean]): STM[E, Option[A]]
```

Finds elements using a transactional predicate.

```scala
val tx: STM[Nothing, Option[Int]] = for {
  arr    <- TArray.make(1, 2, 3, 4, 5)
  ref    <- TRef.make(3)
  result <- arr.findSTM(n => ref.get.map(_ == n))
} yield result
```

## Example: Parallel Updates

```scala
import zio._
import zio.stm._

// Multiple fibers updating array concurrently
val program = for {
  arr <- TArray.make(0, 0, 0, 0, 0).commit
  
  // Launch 5 fibers, each incrementing one element
  fibers <- ZIO.foreach(0 until 5) { i =>
              arr.update(i, _ + 1).commit.repeatN(99).fork
            }
  
  _      <- ZIO.collectAll(fibers.map(_.join))
  result <- arr.toList.commit
  _      <- Console.printLine(s"Final values: $result")
} yield ()
// Output: Final values: List(100, 100, 100, 100, 100)
```

## Example: Atomic Swap

```scala
import zio._
import zio.stm._

// Atomically swap two elements
def swap[A](arr: TArray[A], i: Int, j: Int): USTM[Unit] =
  for {
    vi <- arr(i)
    vj <- arr(j)
    _  <- arr.update(i, _ => vj)
    _  <- arr.update(j, _ => vi)
  } yield ()

val program = for {
  arr <- TArray.make(1, 2, 3, 4, 5).commit
  _   <- swap(arr, 0, 4).commit // Swap first and last
  result <- arr.toList.commit
  _   <- Console.printLine(s"After swap: $result")
} yield ()
// Output: After swap: List(5, 2, 3, 4, 1)
```

## Example: Bounded Buffer

```scala
import zio._
import zio.stm._

class BoundedBuffer[A](private val array: TArray[Option[A]], private val size: TRef[Int]) {
  
  def put(a: A): USTM[Unit] = for {
    s <- size.get
    _ <- STM.check(s < array.size)
    _ <- array.update(s, _ => Some(a))
    _ <- size.update(_ + 1)
  } yield ()
  
  def take: USTM[A] = for {
    s <- size.get
    _ <- STM.check(s > 0)
    _ <- size.update(_ - 1)
    v <- array(s - 1)
    _ <- array.update(s - 1, _ => None)
  } yield v.get
  
  def getSize: USTM[Int] = size.get
}

object BoundedBuffer {
  def make[A](capacity: Int): USTM[BoundedBuffer[A]] = for {
    arr  <- TArray.make(Vector.fill(capacity)(None): _*)
    size <- TRef.make(0)
  } yield new BoundedBuffer(arr, size)
}
```

## Best Practices

1. **Batch updates**: Use `transform` to update all elements atomically
2. **Avoid partial updates**: Complete all related updates in one transaction
3. **Use appropriate size**: Large arrays may increase transaction conflicts
4. **Prefer immutable elements**: Store immutable values for thread safety
5. **Combine with other STM types**: TArray composes well with TRef, TMap, etc.
