---
title: ZSink
---

A `ZSink[R, E, In, L, Z]` is a consumer of values from a stream. Sinks consume values of type `In`, may fail with an error of type `E`, require an environment of type `R`, may produce leftover values of type `L`, and ultimately produce a result of type `Z`.

Sinks are used to aggregate, fold, or otherwise consume stream elements to produce a final result.

## Type Signature

```scala
final class ZSink[-R, +E, -In, +L, +Z] private (
  val channel: ZChannel[R, ZNothing, Chunk[In], Any, E, Chunk[L], Z]
)
```

The type parameters are:
- `R` - Environment required by the sink
- `E` - Error type the sink may fail with
- `In` - Type of values the sink consumes
- `L` - Type of leftover values not consumed
- `Z` - Type of the final result

## Symbolic Operators

- `<*>` - Zip two sinks sequentially
- `<&>` - Zip two sinks in parallel
- `*>` - Sequence sinks, keeping right result
- `<*` - Sequence sinks, keeping left result
- `&>` - Parallel sequence, keeping right result
- `<&` - Parallel sequence, keeping left result
- `|` - Race two sinks
