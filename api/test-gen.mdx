## Overview

The `Gen[R, A]` type represents a generator of values of type `A`, which requires an environment `R`. Generators may be random or deterministic and are the foundation of property-based testing in ZIO Test.

## Type Definition

```scala
final case class Gen[-R, +A](sample: ZStream[R, Nothing, Sample[R, A]])
```

## Core Methods

### Transformation

#### `map`

Transforms generated values with a function.

```scala
def map[B](f: A => B)(implicit trace: Trace): Gen[R, B]
```

**Example:**
```scala
val positiveInts = Gen.int(1, 100)
val doubled = positiveInts.map(_ * 2)
```

#### `flatMap`

Sequentially composes generators.

```scala
def flatMap[R1 <: R, B](f: A => Gen[R1, B])(implicit trace: Trace): Gen[R1, B]
```

**Example:**
```scala
val gen = for {
  length <- Gen.int(1, 10)
  string <- Gen.stringN(length)(Gen.alphaNumericChar)
} yield string
```

#### `mapZIO`

Maps an effectual function over a generator.

```scala
def mapZIO[R1 <: R, B](f: A => ZIO[R1, Nothing, B])(implicit trace: Trace): Gen[R1, B]
```

### Filtering

#### `filter`

Filters generated values. Use sparingly as it can reduce performance.

```scala
def filter(f: A => Boolean)(implicit trace: Trace): Gen[R, A]
```

**Example:**
```scala
val evenNumbers = Gen.int(1, 100).filter(_ % 2 == 0)
```

<Warning>
  Using `filter` can significantly reduce test performance, especially if many values are discarded. Prefer using `map` and `flatMap` to create generators of desired values.
</Warning>

#### `filterNot`

Filters out values that meet the predicate.

```scala
def filterNot(f: A => Boolean)(implicit trace: Trace): Gen[R, A]
```

#### `collect`

Maps values with a partial function, discarding undefined values.

```scala
def collect[B](pf: PartialFunction[A, B])(implicit trace: Trace): Gen[R, B]
```

### Composition

#### `zip`

Combines two generators into a tuple.

```scala
def zip[R1 <: R, B](that: Gen[R1, B])(implicit zippable: Zippable[A, B], trace: Trace): Gen[R1, zippable.Out]
```

**Example:**
```scala
val pairs = Gen.int(1, 10).zip(Gen.string)
// Generates: (Int, String)
```

#### `zipWith`

Combines two generators with a function.

```scala
def zipWith[R1 <: R, B, C](that: Gen[R1, B])(f: (A, B) => C)(implicit trace: Trace): Gen[R1, C]
```

**Example:**
```scala
case class Point(x: Int, y: Int)
val pointGen = Gen.int(0, 100).zipWith(Gen.int(0, 100))(Point.apply)
```

#### `++`

Concatenates two generators.

```scala
def ++[R1 <: R, A1 >: A](that: Gen[R1, A1])(implicit trace: Trace): Gen[R1, A1]
```

### Shrinking

#### `noShrink`

Discards the shrinker for this generator.

```scala
def noShrink(implicit trace: Trace): Gen[R, A]
```

**Example:**
```scala
val gen = Gen.int(1, 1000).noShrink
```

#### `reshrink`

Replaces the shrinker with a simpler one.

```scala
def reshrink[R1 <: R, B](f: A => Sample[R1, B])(implicit trace: Trace): Gen[R1, B]
```

### Size Control

#### `resize`

Sets the size parameter for this generator.

```scala
def resize(size: Int)(implicit trace: Trace): Gen[R, A]
```

**Example:**
```scala
val smallLists = Gen.listOf(Gen.int).resize(5)
```

### Running

#### `runCollect`

Runs the generator and collects all values in a list.

```scala
def runCollect(implicit trace: Trace): ZIO[R, Nothing, List[A]]
```

#### `runCollectN`

Runs the generator N times and collects values.

```scala
def runCollectN(n: Int)(implicit trace: Trace): ZIO[R, Nothing, List[A]]
```

**Example:**
```scala
for {
  samples <- Gen.int(1, 10).runCollectN(5)
  _ <- Console.printLine(s"Generated: $samples")
} yield ()
```

#### `runHead`

Runs the generator returning the first value.

```scala
def runHead(implicit trace: Trace): ZIO[R, Nothing, Option[A]]
```

## Combinators

### `elements`

Generates one of the specified elements.

```scala
def elements[A](as: A*)(implicit trace: Trace): Gen[Any, A]
```

**Example:**
```scala
val color = Gen.elements("red", "green", "blue")
val status = Gen.elements(Status.Active, Status.Inactive, Status.Pending)
```

### `oneOf`

Generates values from one of the specified generators.

```scala
def oneOf[R, A](as: Gen[R, A]*)(implicit trace: Trace): Gen[R, A]
```

**Example:**
```scala
val mixed = Gen.oneOf(
  Gen.int(1, 10),
  Gen.int(100, 1000),
  Gen.int(10000, 100000)
)
```

### `weighted`

Generates values according to weights.

```scala
def weighted[R, A](gs: (Gen[R, A], Double)*)(implicit trace: Trace): Gen[R, A]
```

**Example:**
```scala
// 90% true, 10% false
val biasedCoin = Gen.weighted(
  Gen.const(true) -> 9,
  Gen.const(false) -> 1
)
```

### `const`

A constant generator.

```scala
def const[A](a: => A)(implicit trace: Trace): Gen[Any, A]
```

**Example:**
```scala
val alwaysFive = Gen.const(5)
```

### `fromIterable`

Constructs a generator from an iterable.

```scala
def fromIterable[R, A](
  as: Iterable[A],
  shrinker: A => ZStream[R, Nothing, A] = defaultShrinker
)(implicit trace: Trace): Gen[R, A]
```

**Example:**
```scala
val weekdays = Gen.fromIterable(List("Mon", "Tue", "Wed", "Thu", "Fri"))
```

### `suspend`

Lazily constructs a generator (useful for recursive structures).

```scala
def suspend[R, A](gen: => Gen[R, A])(implicit trace: Trace): Gen[R, A]
```

**Example:**
```scala
sealed trait Tree
case class Leaf(value: Int) extends Tree
case class Branch(left: Tree, right: Tree) extends Tree

lazy val treeGen: Gen[Any, Tree] = Gen.suspend {
  Gen.oneOf(
    Gen.int(1, 100).map(Leaf(_)),
    treeGen.zipWith(treeGen)(Branch(_, _))
  )
}
```

## Property-Based Testing

### Basic Usage

```scala
test("list reverse property") {
  check(Gen.listOf(Gen.int)) { list =>
    assertTrue(list.reverse.reverse == list)
  }
}
```

### Multiple Generators

```scala
test("addition is commutative") {
  check(Gen.int, Gen.int) { (a, b) =>
    assertTrue(a + b == b + a)
  }
}
```

### Custom Generators

```scala
case class Email(user: String, domain: String) {
  override def toString: String = s"$user@$domain"
}

val emailGen: Gen[Any, Email] = for {
  user   <- Gen.alphaNumericStringBounded(3, 20)
  domain <- Gen.elements("gmail.com", "example.com", "test.org")
} yield Email(user, domain)

test("email validation") {
  check(emailGen) { email =>
    assertTrue(email.toString.contains("@"))
  }
}
```
