---
title: ZStream
---

A `ZStream[R, E, A]` is a lazy, concurrent, asynchronous source of values. It represents an effectful stream that requires an environment of type `R`, may fail with an error of type `E`, and produces values of type `A`.

## Type Signature

```scala
final class ZStream[-R, +E, +A] private (
  val channel: ZChannel[R, Any, Any, Any, E, Chunk[A], Any]
)
```

## Core Constructors

### Basic Constructors

#### `apply`


#### `fromIterable`

#### `fromZIO`

## Symbolic Operators

- `>>>` - Pipe through a pipeline or sink
- `++` - Concatenate streams
- `<>` - Fallback to alternative stream on error
- `<*>` - Cross product with another stream
- `<&>` - Zip with another stream
- `*>` - Sequence streams, keeping right result
- `<*` - Sequence streams, keeping left result
