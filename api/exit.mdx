## Overview

An `Exit[E, A]` describes the result of executing a ZIO effect. The result is either succeeded with a value `A`, or failed with a `Cause[E]`.

`Exit` is also a ZIO effect itself, representing a computation that has already completed.

## Type Definition

```scala
sealed trait Exit[+E, +A] extends ZIO[Any, E, A]
```

## Type Parameters

<ParamField path="E" type="type">
  The error type of the exit value
</ParamField>

<ParamField path="A" type="type">
  The success type of the exit value
</ParamField>

## Constructors

### Exit.Success

```scala
final case class Success[+A](value: A) extends Exit[Nothing, A]
```

Represents successful completion with a value.

<ParamField path="value" type="A">
  The success value
</ParamField>

**Example:**
```scala
val exit: Exit[Nothing, Int] = Exit.Success(42)
```

### Exit.Failure

```scala
final case class Failure[+E](cause: Cause[E]) extends Exit[E, Nothing]
```

Represents failed completion with a cause.

<ParamField path="cause" type="Cause[E]">
  The cause of failure
</ParamField>

**Example:**
```scala
val exit: Exit[String, Nothing] = Exit.Failure(Cause.fail("error"))
```

## Pattern Matching

The most common way to work with `Exit` values is through pattern matching:

```scala
exit match {
  case Exit.Success(value) => println(s"Success: $value")
  case Exit.Failure(cause) => println(s"Failure: $cause")
}
```

## Combining Exits

### `<*>` Operator

```scala
def <*>[E1 >: E, B](that: Exit[E1, B]): Exit[E1, (A, B)]
```

Sequentially zips this result with the specified result or else returns the failed `Cause[E1]`.

**Example:**
```scala
val combined = Exit.succeed(1) <*> Exit.succeed(2) // Exit.succeed((1, 2))
```

### `<&>` Operator

```scala
def <&>[E1 >: E, B](that: Exit[E1, B]): Exit[E1, (A, B)]
```

Parallelly zips this result with the specified result or else returns the failed `Cause[E1]`.

### zipWith

```scala
def zipWith[E1 >: E, B, C](
```

<ParamField path="f" type="(A, B) => C">

## Collection Operations

### collectAll

```scala
def collectAll[E, A](exits: Iterable[Exit[E, A]]): Option[Exit[E, List[A]]]
```

Collects all exits into a single exit, succeeding only if all exits succeed.

<ParamField path="exits" type="Iterable[Exit[E, A]]">
  The collection of exits to combine
</ParamField>

**Example:**
```scala
val exits = List(Exit.succeed(1), Exit.succeed(2), Exit.succeed(3))
val combined = Exit.collectAll(exits) // Some(Exit.succeed(List(1, 2, 3)))
```

### collectAllPar

```scala
def collectAllPar[E, A](exits: Iterable[Exit[E, A]]): Option[Exit[E, List[A]]]
```

Collects all exits into a single exit in parallel, combining failures in parallel.

## Usage in ZIO

`Exit` is commonly used when:

1. **Inspecting effect outcomes:**
```scala
for {
  exit <- myEffect.exit
  _    <- exit match {
    case Exit.Success(value) => ZIO.succeed(println(s"Got: $value"))
    case Exit.Failure(cause) => ZIO.succeed(println(s"Failed: $cause"))
  }
} yield ()
```

2. **Working with fibers:**
```scala
for {
  fiber <- task.fork
  exit  <- fiber.await  // Returns Exit[E, A]
  _     <- exit match {
    case Exit.Success(value) => handleSuccess(value)
    case Exit.Failure(cause) => handleFailure(cause)
  }
} yield ()
```

3. **Manual effect execution:**
```scala
import zio.Unsafe

Unsafe.unsafe { implicit unsafe =>
  val exit: Exit[String, Int] = Runtime.default.unsafe.run(myEffect)
  // Handle the exit...
}
```

## Complete Example

```scala
import zio._

object ExitExample extends ZIOAppDefault {
  
  def parseNumber(s: String): IO[String, Int] =
    ZIO.attempt(s.toInt).mapError(_.getMessage)
  
  def run = 
    for {
      // Get exit without failing
      exit1 <- parseNumber("42").exit
      _     <- Console.printLine(s"Result 1: $exit1")
      
      // Pattern match on exit
      exit2 <- parseNumber("not a number").exit
      result <- exit2 match {
        case Exit.Success(n)     => ZIO.succeed(n)
        case Exit.Failure(cause) => ZIO.succeed(0) // Provide default
      }
      _     <- Console.printLine(s"Result 2: $result")
      
      // Fold over exit
      message <- parseNumber("123").exit.map(_.foldExit(
        cause => s"Failed: $cause",
        value => s"Parsed: $value"
      ))
      _       <- Console.printLine(message)
      
    } yield ()
}
```
